このファイルは、私(S_hiba)のポートフォリオサイトのソースコードをまとめたものです。
内容は下記の通りです。

・source
ポートフォリオサイトのソースコード本体です。
PHP、HTML(Smartyテンプレート)、CSS(Sassファイル)、画像、その他設定ファイルが含まれています。
※サイト内のアイコンには、「Font Awesome」を利用しています。
Font Awesome: https://fontawesome.com/
※言語などのロゴ画像は、「SVG PORN」のSVGファイルをpng画像に変換して利用しています。
SVG PORN: https://svgporn.com/

・sql
DBテーブルのCREATE文や、デフォルトデータのINSERT文をSQLファイルとしてまとめてあります。
